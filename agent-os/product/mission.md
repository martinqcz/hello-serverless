# Product Mission

## Problem

Learning and experimenting with AWS serverless architecture requires a working reference project that demonstrates real-world patterns and deployment workflows.

## Target Users

Personal use — a reference implementation for AWS serverless development patterns.

## Solution

A complete full-stack serverless reference: Kotlin/Micronaut backend with GraalVM native compilation, Vue 3 frontend with Vuetify, and automated AWS deployment scripts. Demonstrates the entire stack from local development to production deployment.
