# Product Roadmap

## Phase 1: MVP (Complete)

- Basic REST API endpoint (`/api/v1/hello`)
- Vue 3 + Vuetify frontend with i18n and theme switching
- DynamoDB integration with LocalStack for testing
- GraalVM native Lambda compilation
- Multi-environment deployment scripts (dev/prod)
- CloudFront + S3 frontend hosting

## Phase 2: Post-Launch

- **Auth (Cognito)** — User authentication and authorization
- **CRUD operations** — Full DynamoDB entity management with create/read/update/delete
